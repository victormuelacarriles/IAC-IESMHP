# Cambios 2026-06-08

## 09:01 — Rol `nvidia`: instalar NVIDIA Container Toolkit si hay GPU

**Fichero**: `Ubuntu/ansible/roles/nvidia/tasks/main.yml` (+ `CLAUDE.md` del rol).

Al final del bloque `when: has_nvidia` (tras comparar el driver recomendado
con el instalado) se añade la instalación del **NVIDIA Container Toolkit**,
solo si hay GPU NVIDIA y solo si no está ya instalado:

1. `dpkg-query` comprueba si `nvidia-container-toolkit` ya está instalado
   (`changed_when: false`, `failed_when: false`).
2. Bloque `when: "'install ok installed' not in nct_installed.stdout"`:
   - Descarga la clave GPG oficial a
     `/usr/share/keyrings/nvidia-container-toolkit-keyring.gpg`
     (`creates:` para no re-descargar).
   - Añade el repo APT a
     `/etc/apt/sources.list.d/nvidia-container-toolkit.list` con `sed`
     para inyectar `signed-by=` (`creates:` idempotente).
   - `apt install nvidia-container-toolkit` con `update_cache: yes` (repo
     recién añadido).
3. `nvidia-ctk cdi generate --output=/etc/cdi/nvidia.yaml` con
   `creates: /etc/cdi/nvidia.yaml` y `failed_when: false` (en instalación
   fresca el módulo NVIDIA puede no estar cargado aún; el reinicio final de
   `3-SetupPrimerInicio.sh` lo activa).

Permite que Docker/Podman accedan a la GPU vía CDI. Idempotente.
