# Rol `notepadpp`

## Qué hace
Instala **Notepad++** con Chocolatey (`notepadplusplus`), fijando `notepadpp_version`
(por defecto `8.9.6.4`).

## Estructura
- `tasks/main.yml`
- `defaults/main.yml` → `notepadpp_version` (vaciar para la última).

## Notas
- Alternativa: winget `Notepad++.Notepad++`.
